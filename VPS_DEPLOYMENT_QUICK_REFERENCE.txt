## Alternative: Manual Deployment

### On VPS:
```bash
ssh root@YOUR_VPS_IP

# Navigate to app directory
cd /var/www/yamaha

# Stop current server
pm2 stop yamaha-game
pm2 delete yamaha-game

# Start SQLite server
pm2 start server-sqlite.js --name yamaha-game
pm2 save

# Check logs
pm2 logs yamaha-game
```
